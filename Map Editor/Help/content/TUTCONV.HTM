5. Converting Single And Multiplayer Maps

FinalAlert 2: Yuri's Revenge TM allows you to convert singleplayer maps to multiplayer maps and vice versa. To convert from singleplayer to multiplayer:

1) Click on File->New
2) Multiplayer map
3) Import Map
4) Browse for your map
5) Click OK.
6) Save the map as .mpr in the RA2 directory

FinalAlert 2: Yuri's Revenge TM will now automatically repair the map and produce a multiplayer map. Please note that any building and unit on the map should be given to the neutral player (or special player). FinalAlert 2: Yuri's Revenge TM does not do this automatically.

Converting multiplayer maps to singleplayer maps works the same way, but don’t check the multiplayer option, and make sure you save as .map file.
