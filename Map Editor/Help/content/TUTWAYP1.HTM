6. Waypoints tutorial (beginners)

Creating & Using Waypoints
Waypoints are "flags" used to show special locations for occurring events such as arriving reinforcements, where to move the camera (in multiplayer maps) or to define the starting locations of the players. Waypoints are created and deleted using the object browser on the left side of the screen.

Step 1: Creating A Waypoint
Resize the browser to see the all the list options. Two-thirds of the way down, you will see the "Waypoints" list item. Click on the "+" to open the options for waypoints.

Step 2: Choose The Type
In the list are 3 options: "Create Waypoint", "Create Waypoint with Special ID", and "Delete Waypoint." The first option will add a waypoint of the next available number to the map, the second gives it any number you wish, and the third deletes any waypoint on the map. Select the second option, "Create Waypoint with Special ID".

Step 3: Set The Waypoint
Click anywhere on the iso view. A dialog box comes up. Give the waypoint the number value "60". This special number value will be used to identify the waypoint in the future. Click OK. An icon of a flag now appears at the spot you clicked with the number you designated.

Deleting Waypoints
Open up the "Waypoints" list in the object browser and select "Delete Waypoint." Now click on the waypoint you wish to remove.

Note: You cannot use "Delete Object" to delete a waypoint.

Searching Waypoints
If you need to search for a specific waypoint on the map, go to the Map tools menu and select Search Waypoint. A dialog will pop up and you can select the waypoint that shall be centered in the IsoView.
