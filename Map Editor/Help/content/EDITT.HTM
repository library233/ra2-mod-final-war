5. Editing Terrain

You can access the terrain editing features using the "Terrain" and "Map tools" menu or using the terrain toolbar. Below you’ll find the terrain toolbar and special tools bar:

Note: #8 and #9 are not recommended and therefore not documented below.

1. Raise Ground: Click on Raise Ground, and then on the IsoView where you want to raise the ground on your map. The user interface is fairly self-descriptive, but you’ll have to take care of some important things. Try out the terrain editing tutorial for those. If you use features #1 or #2 on "non-morphable" terrain (terrain that is always plain or flat, like water, streets and cliffs), then the following facts apply:

a) CTRL key not pressed while clicking: The complete tile will be raised without modifying any other tile. For example if you used the terrain browser to place some cliff parts (=tiles), only one of these parts will be raised.

b) Pressing down CTRL key while clicking: Whole surface filled with this tileset/terraintype will be raised, this may take some seconds! Please do not boot FinalAlert 2: Yuri's Revenge(tm), as this may take quite long if the surface is big!


2. Lower Ground: Lower ground works like raise ground (even for non-morphable terrain and CTRL key).

3. Flatten Ground: Flatten ground uses the height of the tile you clicked on when last you clicked the mouse button. It also sets every tile you move to the height of this tile if you hold down the mouse button. Flatten ground does not raise "non-morphable" terrain but it may use "non-morphable" terrain to choose the height. Uses brush size

4. Hide Tileset: Hide tileset can be used to hide map features. For example, cliffs can be hidden so that you can access the ground behind the cliff with your mouse. Just click on a cliff and the whole cliff (and every other cliff) will be made invisible. Works with every terrain type. Instead of the graphic a small green rect is shown to represent the fields (but they will not be accessible with your mouse).

5. Show All Tilesets: Shows every tileset.

6. Hide Single Field: Works like #4, but only hides one single field of the map.

7. Show All Fields: Shows every field hidden by #6.

8. Raise Single Tile: This feature is not documented. Use at your own risk.

9. Lower Single Tile: This feature is not documented. Use at your own risk.

10. Framework mode: Enables Framework mode. This mode will make the IsoView to display tiles like cliffs using representating graphics that allow you to judge the map appearance much easier.

11. Paint Cliff Front: Select this tool to make cliff fronts and then move your mouse into the IsoView where you want your cliff to begin. Hold down the mouse button and then move the mouse to the end of the cliff. Release the button when the cliff is where you want it to be

12. Paint Cliff Back: Same like #10, but for the back of cliffs.

Note: In the New Urban theater, there are 2 cliff types. To switch between those 2 cliff types, press the "A" key.


13. Autolevel: AutoLevel is just an icon for the MapTools->AutoLevel menu described above.

14. Autoshore: AutoShore is just an icon for the MapTools->AutoShore menu described above

Painting Terrain and Placing Tiles and Overlays

This feature allows you to place the different terrain types like cliffs, water, sand, tunnel entrances or waterfalls. It also allows you to place overlays.

If you want to place terrain/tiles, select the type of terrain you want to place in the left window on the Main Screen. After you selected a folder, many different pictures will appear at the bottom of the screen. Click on a picture to select it and then move your mouse into the IsoView. Move it to where you want it to be, and click. If you hold down the shift button you can move your mouse around and place many tiles at once. Only do this if you want to fill a large area with similar tiles - for example, water. If you hold down the CTRL key while placing the tile, the area of the map will be filled with the tile. This is useful for placing water also. Uses brush size

You can use the Page Up and Page Down keys to change the tile placement height.

You can use:
MapTools->AutoLevel to automatically raise the ground inside a cliff
MapTools->Paint cliffs to paint cliff fronts and backs using a drag system
MapTools->AutoShore to automatically create shores that connect grass and water.


Using Arrow Keys to Navigate Tilesets

You can scroll through the tiles and tilesets using the arrow keys on your keyboard. First select any tile in the terrain browser to activate the terrain painting mode. Now, move the mouse onto the map and press:

This is useful for placing cliffs, water, and things like that.

If you want to place overlay (like Ore and mid parts of bridges), select the overlay you want in the right box then continue like with placing terrain/tiles.

If you made a mistake, you can use Edit->Undo to reverse the last changes made to the terrain and overlays (not units and objects).

You can use Edit->Copy to copy parts of the map into the clipboard and use Edit->Paste to paste them into the map. After clicking on Edit->Copy, click on 2 points in the map that define the region to be copied, then Edit->Paste to activate the paste mode. Be aware that this even works for different maps and between FinalSun<->FinalAlert 2: Yuri's Revenge TM. You can use the page up and page down keys to change the height of the inserted area.
